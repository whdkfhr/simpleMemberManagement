# simpleMemberManagement

### 회원 관리 예제

#### 간단한 예제를 통해 스프링의 흐름 

+ 비즈니스 요구사항 정리
  + 데이터 : 회원 ID, 이름
  + 기능 : 회원 등록, 조회
  + 아직 데이터 저장소가 선정되지 않음
  + 컨트롤러 : 웹 MVC의 컨트롤러 역할
  + 서비스 : 핵심 비즈니스 로직 구현
  + 리포지토리 : 데이터베이스에 접근, 도메인 객체를 DB에 저장하고 관리
  + 도메인 : 비즈니스 도메인 객체. 예) 회원, 주문, 쿠폰 등등 주로 데이터베이스에 저장하고 관리됨
+ 회원 도메인과 레포지토리 만들기
+ 회원 리포지토리 테스트 케이스 작성
+ 회원 서비스 개발
+ 회원 서비스 테스트
